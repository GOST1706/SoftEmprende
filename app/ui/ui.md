# Carpeta /ui
Maneja todo lo visual del sistema.

## Lo que vive aquí
- Ventanas del admin
- Ventanas del cajero
- Modales / paneles (agregar producto, registrar gasto, etc.)
- Componentes reutilizables (botones, inputs, tablas)

## Objetivo
Separar la interfaz para que no mezcle lógica de negocio.
