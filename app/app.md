# Carpeta /app
Contiene todo el código principal del programa.

## Estructura general
- /ui     → interfaces gráficas, ventanas y mockups funcionales.
- /core   → lógica del negocio (ventas, inventario, compras, gastos).
- /data   → manejo de base de datos local y API remota.
- /sync   → sincronización de datos entre local y nube.
- /utils  → funciones auxiliares y helpers.
- main.py → punto de arranque de la app.
